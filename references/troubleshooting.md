# Troubleshooting

- `E120_ENVIRONMENT_MISMATCH`: use administrator Python 3.10 on Windows and verify the Origin-side Python modules.
- `E121_ATTACH_POLICY_VIOLATION`: start a visible Origin instance as administrator before a formal run.
- `E131_TEMPLATE_RETRIEVAL_EXHAUSTED`: do not infer an outage from one client or parser error. After a failure, make at least three total attempts for that official URL unless an earlier retry succeeds; switch between direct HTTP retrieval and an available browser/client, inspect raw HTML for official GID/download links, try alternate close Gallery candidates, validate ZIP/SHA256, and check both the workspace catalog and installed templates. Keep the attempt log. If every recorded route fails, report the external limitation and use a native reconstruction only when the route contract permits it.
- Locked OPJU: close the project, check the exact `.lck`, and inspect residual hidden Origin processes before retrying. Never batch-delete files.
- Modal dialog: resolve it without changing the privilege envelope; do not treat a blocked export as a pass.
- Demo watermark: invalidate the whole run, detach in `finally`, preserve only failure diagnostics, and restart once from `assert_admin_preflight.py` with a new run ID and clean output root under administrator Python plus a visible administrator-started Origin. Never continue from save/reopen/QA or reuse the watermarked run's artifacts. If the fully elevated restart is also watermarked, stop and resolve the Origin license/environment; do not loop.
- Zero plot readback: compare `plot_list()` with `layer -c`, activate the layer, and inspect the live binding chain.
- Blank export: reject before visual scoring; verify page/layer units, visibility, data ranges, and post-reopen export.
- Two identical correction iterations: stop with `E510_NO_IMPROVEMENT` and inspect the dominant mismatch rather than changing unrelated parameters.
- Fig16 excess dark-blue pixels: inspect group-frame width before changing bar colors.
- Fig16 H/S overlap: calibrate the affected stage locally; do not apply a global shift that damages other stages.
