# Evidence Contract

## Basic

Produce the OPJU, reopened Origin export, and summary. Record profile, mode, enabled gates, command status, structure status, live status, warnings, and errors.

## Visual

Add reopened object/binding readback, registered visual metrics, and a manifest. Do not create a full release evidence directory by default.

## Full

Add pre-save and post-reopen Origin exports, complete readback, visual metrics, provenance, SHA256 records, route and source identities, template search evidence, evidence directory, release status, and package/batch audits.

Every manifest must mark disabled gates `not_required`. Never copy `pass` from an old manifest. Derive live and release status only from same-run artifacts.
