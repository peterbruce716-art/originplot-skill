# Legacy and Migration

V2, V3, and V4 artifacts are legacy inputs. The source tree retains some V4 scripts and examples for explicit migration and regression work, while the p13/p14/p15 shareable package excludes them. They are not the current runtime and must not be presented as supported live entry points.

Current execution uses V5 schemas and the p15 candidate worker. A legacy artifact must be validated, converted explicitly, and then pass current route, provenance, reopen, structure, and visual gates. Migration does not grant pass eligibility.

Superseded routes include Matrix-only final Fig12 contour, GraphObject-only Fig16 geometry, zero-plot acceptance, copied seed OPJUs, source-image pages, pixel canvases, and worksheet inventories without direct plot readback. Preserve them only as labelled diagnostics where needed for research history.

The p13-to-p14 instruction mapping is recorded in `../p14_gap_analysis.md`.
