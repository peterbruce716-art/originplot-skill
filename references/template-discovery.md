# Template Discovery

## Skip

Do not search locally or online. Use an explicit native builder. This is the Quick default.

## Auto

Search local installed templates or the workspace catalog first. If still useful, perform one bounded official Gallery search and inspect at most three semantically close candidates. Record a compact candidate list and selection/rejection reasons. A network or parser failure is a warning; continue with native reconstruction.

Never treat a screenshot, filename, `.otp`, or download as evidence that a project is reusable. A template is reusable only after compatible Origin inspection confirms editable graphs, data, and bindings.

## Strict

Use only for Release. Follow all official/local entrances, retry and archive validation rules, SHA256 checks, safe extraction, administrator Origin editable-open inspection, and the complete template search record. Missing evidence fails with `E130_TEMPLATE_SEARCH_REQUIRED` or the more specific retrieval error.
